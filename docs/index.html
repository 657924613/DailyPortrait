功能

简单到只需一个动作

📸
每日拍照
前置摄像头 + 洋葱皮叠加 + 实时对齐引导环，让每张照片里你的脸在同一位置。
🎬
延时视频
帧率、画质、仅对齐照片可调。一键导出到系统相册，分享到任何平台。
🔔
定时提醒
自定义提醒时间，每天准时推送通知。今天已打卡就不打扰。
🔥
连续打卡
自动计算连续天数，断签清零。周进度 7 天圆点一目了然。
↩️
可撤销删除
误删不慌。5 秒内点撤销即可恢复，超时才真正删除文件。
🌙
深色模式
跟随系统自动切换。暖深棕背景，不是纯黑，保持品牌调性。

隐私是底线，不是卖点

你的照片是你最私密的数据。我们从第一行代码就把隐私当作不可妥协的约束。

  • 所有照片仅存设备沙盒，卸载即清
  • ML Kit 静态版，零网络请求
  • 无广告、无追踪、无用户账号
  • 不收集 IMEI / 位置 / 通讯录
  • 视频合成完全本地，不上传任何服务器
  • 代码完全开源，接受任何人审计
