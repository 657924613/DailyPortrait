## 十、技术说明

- **Stack**：单文件 HTML + React 18（UMD）+ Babel Standalone
- **无外部依赖**：双击 `.html` 即可在浏览器打开
- **字体**：`-apple-system` + `SF Pro Text` + `PingFang SC`（系统默认）
- **动画**：纯 CSS `@keyframes` + React `useState` 驱动的 className 切换；遵循 prefers-reduced-motion
- **可访问性**：语义 HTML（`<main>` `<nav>` `<section>` `<aside>`）+ ARIA role 标注 + 色盲友好 ✓/✗ 图标 + 对照 WCAG 2.2 AA 清单（见 §6）
