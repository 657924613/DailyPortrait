## 二、我们收集的信息

### 2.1 您主动提供的信息

| 类型 | 用途 | 是否必需 |
|------|------|---------|
| 自拍照片 | 记录每日肖像、生成延时视频 | 是 |
| 拍摄日期与时间 | 排序与连续打卡统计 | 是 |
| 提醒时间偏好 | 在指定时间发送通知 | 否（可关闭） |

### 2.2 自动收集的信息

| 类型 | 用途 | 是否必需 |
|------|------|---------|
| App 版本号 | 故障排查 | 是 |
| 系统版本 / 设备型号 | 兼容性优化（仅崩溃时上传） | 否 |
| 崩溃堆栈 | 定位 bug | 否（首次启动可关闭） |

**我们不会收集的信息（明确承诺）**：
- ❌ 您的真实姓名、身份证号、手机号
- ❌ 通讯录、位置、相册中其他照片
- ❌ 设备 IMEI / IDFA / Android ID
- ❌ 您在其他应用中的行为数据

## 三、信息存储与处理

### 3.1 本地存储

**所有照片仅存储在您设备的应用沙盒中**：
```
/Android/data/com.missyun.dailyportrait/files/Pictures/
/Android/data/com.missyun.dailyportrait/files/Movies/
```

卸载应用即彻底删除，**不留服务端备份**。

### 3.2 人脸识别（ML Kit）

我们使用 Google ML Kit 静态版本进行**本地**人脸识别：
- ✅ 完全离线，照片不会上传到 Google 或任何第三方服务器
- ✅ ML 模型已打包在 APK 内，启动后不下载
- ✅ 仅用于辅助构图对齐，不识别身份

### 3.3 视频导出

延时视频在您本机生成与保存，**全程不联网**。
您可通过系统分享功能自行决定是否上传到第三方平台（微信 / 抖音等），
分享行为本身受目标平台隐私政策约束，不在本政策范围内。

## 四、第三方服务

| 服务 | 用途 | 数据范围 |
|------|------|---------|
| Google ML Kit Face Detection | 本地人脸识别 | 仅本地处理，零数据上传 |
| Sentry（如启用） | 崩溃报告 | 仅崩溃堆栈，不含照片 / 用户内容 |

## 五、您的权利

根据《中华人民共和国个人信息保护法》、《GDPR》（如适用）等法规，您拥有：

1. **查看权**：所有数据均在您设备上，可直接通过应用查看
2. **删除权**：在 Dashboard 长按任意照片即可删除；卸载应用清除全部
3. **撤回授权权**：在系统设置中关闭相机 / 通知权限
4. **导出权**：通过系统文件管理器直接复制 `Android/data/com.missyun.dailyportrait/` 整个目录
5. **投诉权**：联系我们或向当地数据保护机构投诉

## 六、未成年人保护

本应用面向 12 岁以上用户。未成年人请在监护人陪同下使用。
若发现未成年人未经同意拍摄照片，监护人可通过卸载应用清除全部数据。

如本政策有重大变更，我们会在更新版本中以显著方式提示。
您持续使用即视为同意更新后的政策。
